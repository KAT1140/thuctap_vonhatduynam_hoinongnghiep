Nhập Hội Viên Hàng Loạt

Nhập tất cả file Excel từ thư mục "exel hội viên theo từng ấp"

⚠️ Xác nhận nhập dữ liệu hàng loạt

Thao tác này sẽ nhập tất cả các file Excel từ thư mục vào hệ thống.

Hệ thống sẽ:

  • Quét tất cả file Excel (.xlsx, .xls) trong thư mục "exel hội viên theo từng ấp"
  • Trích xuất tên ấp từ tên file (ví dụ: "hội ấp bà trầm.xlsx" → ấp "bà trầm")
  • Tìm tổ hội tương ứng với ấp đó
  • Nhập dữ liệu từ mỗi file

Thư mục:

exel hội viên theo từng ấp/

Cấu trúc file Excel:

  1. Họ tên (bắt buộc)
  2. Ngày sinh
  3. Giới tính
  4. Căn cước/CMND
  5. Điện thoại
  6. Email
  7. Địa chỉ
  8. Trình độ
  9. Dân tộc
  10. Tôn giáo
  11. Phân loại
  12. ID Tổ hội (tuỳ chọn)

💡 Lưu ý:

Nếu không có ID Tổ hội, hệ thống sẽ tự động tìm tổ hội dựa vào tên ấp.
