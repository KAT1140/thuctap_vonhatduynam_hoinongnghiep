Nhập Hội Viên từ File Excel

Tải lên file Excel để nhập danh sách hội viên hàng loạt

Tải lên file Excel
Chỉ chấp nhận file có định dạng .xlsx hoặc .xls. Tối đa 16MB.
Quay lại
Hướng dẫn
Cột trong file Excel:
  1. A: Họ tên (bắt buộc)
  2. B: Ngày sinh (định dạng: YYYY-MM-DD)
  3. C: Giới tính (nam/nữ/khác)
  4. D: Căn cước/CMND (bắt buộc, không trùng)
  5. E: Điện thoại
  6. F: Email
  7. G: Địa chỉ
  8. H: Trình độ (thieu_hoc/tieu_hoc/trung_hoc/cao_dang/dai_hoc)
  9. I: Dân tộc
  10. J: Tôn giáo
  11. K: Tên ấp
  12. L: Phân loại (thuong/dang_vien/nong_cot)
  13. M: ID Tổ hội (bắt buộc)
Lưu ý:
  • Dòng đầu tiên phải là tiêu đề cột
  • Họ tên và ID Tổ hội không được để trống
  • ID Tổ hội phải tồn tại trong hệ thống
  • Để to_hoi users chỉ được nhập vào próprio tổ của mình
